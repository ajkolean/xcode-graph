# Utilities

Helper components and tools that enhance the user experience.

## ⌨️ KeyboardShortcuts

Interactive help dialog showing all keyboard shortcuts available in the application.

### Features
- Toggle button to show/hide shortcuts
- Modal overlay with shortcut list
- Keyboard-accessible (press `?` to open)

### Common Shortcuts
While the component displays all shortcuts, typical ones include:
- **Navigation**: Arrow keys, Tab
- **Zoom**: `+` / `-` / `0` (reset)
- **Selection**: Click, Shift+Click
- **Search**: `/` to focus search
- **Help**: `?` to show shortcuts

### Usage
This component is typically rendered at the app level and controlled by keyboard events.
