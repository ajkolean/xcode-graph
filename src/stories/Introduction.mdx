# TuistGraph Component Library

Welcome to the TuistGraph Storybook! This is your interactive guide to all UI components used in the dependency graph visualization application.

## 🏗️ Architecture Overview

TuistGraph is built with **Lit Web Components**:

- **Lit Web Components**: Performant, framework-agnostic UI components with Shadow DOM encapsulation
- **Pure Web Standards**: Built on standard Custom Elements and Shadow DOM APIs

This architecture provides:
- ⚡ **Performance**: Lit's lightweight runtime and reactive system
- 🔄 **Reusability**: Web components work across any framework
- 🎨 **Encapsulation**: Shadow DOM prevents style conflicts
- 🧪 **Testability**: Component tests with accessibility and interaction validation

## 📚 Component Categories

### Design System
Core UI building blocks used throughout the application:
- **UI Components**: Buttons, inputs, cards, lists
- **Icons**: Filter section icons, navigation icons
- **Badges & Labels**: Type indicators, status badges

### Features
Feature-specific component collections:
- **Node Details**: Components for displaying individual node information
- **Cluster Details**: Components for displaying cluster/project information
- **Filters**: Search and filter interface components

### Graph Visualization
SVG-based graph rendering components:
- **Nodes & Edges**: Core graph elements
- **Clusters**: Project/package grouping visualization
- **Overlays**: Controls, backgrounds, empty states

### Layout
Application layout and navigation:
- Headers, sidebars, toolbars

### Panels & Views
High-level container components that orchestrate child components

### Utilities
Helper components and tools

## 🧪 Testing Patterns

Each component includes:
- **Playground**: Interactive controls for experimentation
- **State Variants**: Different props and states
- **Interactive Tests**: Automated play functions testing events and interactions
- **Accessibility Tests**: Automated a11y validation with WCAG compliance

## 🎯 Using This Storybook

1. **Browse by category** in the sidebar to explore components
